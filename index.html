Multi-sensoric vr controller

Goal for this research project was to build an alternative controller for VR environments.

I built a controller prototype with Arduino sensor modules, such as ultrasonic, sound, gyroscope and accelerometer. Then I processed the sensor data in a C++ application which communicated to a Unity application.
